# Privacy Policy for Kick SmoothView

Last updated: 2026-05-02

Kick SmoothView is a Chrome extension designed to reduce chat lag on Kick.com and apply a user's preferred stream quality.

## Data collection

Kick SmoothView does not collect, transmit, sell, or share personal data.

The extension stores settings locally in the user's browser using Chrome storage. These settings may include:
- selected extension mode
- whether emotes/images/animations are hidden
- maximum chat messages kept
- preferred stream quality
- selected interface language

These settings stay on the user's device and are not sent to any server.

## Website access

Kick SmoothView runs only on Kick.com pages. It modifies the local browser page to reduce heavy chat elements and apply stream quality preferences.

## Third-party sharing

Kick SmoothView does not share data with any third party.

## Remote code

Kick SmoothView does not load or execute remote code. All extension code is included in the extension package.
