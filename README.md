# Kick SmoothView

Release UI build.

Features:
- Reduce Kick chat load
- Hide emotes/images
- Disable animations
- Limit old chat messages
- Apply preferred stream quality
- Auto-save settings
- Arabic / English UI
- ON badge while active on Kick

Important notice:
Some gift or subscription alerts may not appear when Hide emotes or Hide chat images is enabled.

Debug:
